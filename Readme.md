// jwt ek library hai jo JSON Web Tokens (JWT) ko create, sign, verify, aur decode karne ke liye use hoti hai. JWT ek compact, URL-safe token format hai jo client-server communication mein authentication aur authorization ke liye use hota hai. JWT tokens mein user ki information encoded hoti hai, jisse server verify kar sakta hai ki token valid hai ya nahi, aur uske basis par user ko access dene ya deny karne ka decision le sakta hai.

// bcrypt ek library hai jo password hashing ke liye use hoti hai. Password hashing ek security technique hai jisme user ke password ko ek irreversible format mein convert kiya jata hai, taki agar database compromise ho jaye to attackers ko original passwords nahi mil sake. Bcrypt hashing algorithm salt (random data) ka use karta hai, jisse same password ke liye bhi different hashes generate hote hain, aur brute-force attacks se bachav hota hai.
